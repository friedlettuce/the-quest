"Lost Jungle"
by Eric Matyas
www.soundimage.org

"Ambience, Day Wildlife, A.wav"
by InspectorJ
(www.jshaw.co.uk) of Freesound.org

##### How to Play
- Right arrow : move right
- Left arrow : move left
- Q : use weapon
- V: toggle weapon
###### (Wizard)
- W : use spell
- E : switch spell
